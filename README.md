# RentCar SaaS

A React-based car rental SaaS application built with Parcel bundler.

## Project Structure

- `index.6b53a5f6.js` - Production-ready minified JavaScript bundle

## Technologies Used

- React (Frontend framework)
- Parcel (Build tool)
- FontAwesome Icons

## Development Notes

This repository currently contains the production build. For development:
1. Consider adding source files in a `src/` directory
2. Add `package.json` with dependencies
3. Add `.gitignore` to exclude node_modules and build artifacts
4. Keep source maps separate or in a dedicated branch

## Getting Started

To use this application, serve the JavaScript file with an HTML page that includes:
```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>RentCar SaaS</title>
</head>
<body>
    <div id="react"></div>
    <script src="index.6b53a5f6.js"></script>
</body>
</html>
```
